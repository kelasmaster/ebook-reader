Selamat datang di eBook tentang HTML, CSS, dan JavaScript untuk pemula. Dalam bab ini, kita akan membahas dasar-dasar pengembangan web.

Apa Itu Web Development?

Web development adalah proses membuat aplikasi atau situs web yang dapat diakses melalui internet. Ini mencakup:

  • Frontend: Bagian yang terlihat oleh pengguna.
  • Backend: Bagian yang berjalan di server.

Tools yang Dibutuhkan

  • Text Editor (VS Code, Sublime Text)
  • Browser (Chrome, Firefox)
  • Dasar Pengetahuan HTML, CSS, dan JS
