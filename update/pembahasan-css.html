Bab 3: Pembahasan CSS

CSS (Cascading Style Sheets) digunakan untuk mendesain halaman web.

Contoh Penggunaan CSS


body {
  background-color: #f4f4f4;
  font-family: Arial, sans-serif;
}

h1 {
  color: #333;
}

Manfaat CSS

  • Memisahkan konten dari desain.
  • Membuat halaman lebih responsif.
  • Menghemat waktu dengan reusable styles.
